# Security Policy

## Reporting Vulnerabilities

We take security seriously. If you discover a security vulnerability, please report it to us responsibly by sending an email to **info@nma.vc**.

**Please do NOT open public issues for security vulnerabilities.**

## Supported Versions

Only the latest minor release of `takeln` is supported with security updates.

## Response Timeline

- We aim to respond to vulnerability reports within **48 hours**.
- For critical issues, we aim to provide a fix within **7 days**.

Thank you for helping keep `takeln` and its users safe. Your responsible disclosure is greatly appreciated.
